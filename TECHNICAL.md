# Zhetysu Bayi Steel — техническая документация

## Назначение
Одностраничный русскоязычный корпоративный сайт-каталог официального дистрибьютора Jingye Steel Group в Казахстане. Структура ориентирована на B2B-запросы и отправку спецификаций.

## Структура
- `app/page.tsx` — первый экран, быстрый запрос, товарный каталог, производитель, преимущества, процесс поставки и контактный CTA.
- `app/globals.css` — визуальная система и адаптивные стили.
- Блок преимуществ использует CSS-пиктограммы, короткие метки и hover-состояния без внешней библиотеки иконок.
- `public/images/` — локальные изображения с официального сайта Jingye Steel Group.
- `.openai/hosting.json` — конфигурация публикации в Sites.

## Контент и контакты
Фотографии и факты о производителе взяты с `https://www.hebeijingyesteel.com/about_us.html`.
Перед финальным запуском подтвердить рабочий email. Сейчас CTA использует `info@zhetysubayisteel.kz` как предполагаемый адрес.

## Запуск
`npm run dev` — локальная разработка. `npm run build` — проверка production-сборки.
